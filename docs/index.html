IHY

Self-hosted music · ancient soul

Your music, on your hardware, named after the Egyptian god of music. A fast multi-user web player with gapless playback, a full tag editor, smart playlists, automatic downloads — and an API your phone already speaks.

Install with Docker GitHub ↗
Features

Everything a player should do

Built for people who keep their own files — and expect the polish of a streaming service.

Playback

  • Gapless + crossfade (3–12s)
  • 10-band EQ, speed, sleep timer
  • ReplayGain volume normalization
  • Autoplay radio at queue end
  • Synced karaoke lyrics
  • Cross-tab control & Chromecast

Library

  • Artists, albums, genres, folders
  • Full tag editor, single & batch
  • Multi-artist tag splitting
  • Smart playlists with rules
  • M3U / XSPF import & export
  • Covers, artist images & bios

Integrations

  • OpenSubsonic API for mobile apps
  • ListenBrainz & Last.fm scrobbling
  • spotdl watches on a schedule
  • Opus transcoding with cache
  • Lyrics from lrclib.net

Platform

  • Installable PWA, offline downloads
  • Responsive mobile UI
  • Command palette (Ctrl+K)
  • Multi-user, backup & restore
  • HTTPS out of the box
Install

One image, two minutes

terminal
# clone and configure
git clone https://github.com/n36l3c7/Ihy.git
cd Ihy/docker
echo "IHY_SECRET_KEY=$(openssl rand -hex 32)" > .env

# mount your music in docker-compose.yml, then
docker compose up -d --build

# first run: create the admin account and scan your folders
https://localhost:8000

Needs Docker Engine 24+ and ~1 GB of RAM. Python, Node, ffmpeg, spotdl and a self-signed HTTPS certificate are all handled inside the image. Full details in the README.
